ChatGLM 最初以中英双语对话模型形态出现，后续在参数规模、对齐方法与工具生态上持续迭代，推出了多代模型与开源版本。它在中文问答、办公助手、知识库问答等场景中有较多落地案例，并与智谱自家的 API、知识库与 Agent 平台深度整合。

对于需要在国内环境中部署、兼顾 API 与私有化选项的团队，ChatGLM 系列是重要的候选模型之一。
