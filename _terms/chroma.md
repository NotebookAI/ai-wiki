Chroma 提供简单的 Python API 与持久化机制，让用户在本地或单机环境快速构建向量索引和元数据存储。它支持多种嵌入模型和过滤条件，适合笔记管理、文档助手等中小规模场景。

在工程实践中，Chroma 常被用作 RAG 系统的早期原型或个人知识库的后端，后续可迁移到 Milvus、Pinecone 等更大规模的服务。
