GPT-4 系列是 OpenAI 发布的大语言模型家族，相比 GPT-3.5 在推理能力、对齐质量与安全性上显著提升。GPT-4o（omni）进一步统一了文本、图像、音频等多模态能力，强调「端到端多模态」「低延迟」与更亲民的价格，使其适合嵌入到各种应用之中。

在产品形态上，GPT-4/4o 既以 ChatGPT 形式面向终端用户，也通过 OpenAI API 提供模型、工具调用（Function Calling）、文件与知识库检索、Assistants API 等能力，是许多 Agent、RAG、自动化工作流系统的底座之一。
